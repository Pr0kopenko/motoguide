# ВИРАЖ

Кураторский одностраничный мотогайд по кольцевым маршрутам одного дня из Москвы.

- Сайт: <https://virazh-motoguide.pr0kopenko.chatgpt.site>
- Исходный код: <https://github.com/Pr0kopenko/motoguide>

В первой версии есть:

- 6 маршрутов без Переславля;
- фильтры по длительности и характеру поездки;
- подробный roadbook выбранного маршрута;
- предупреждения о нагрузке и покрытии;
- готовые цепочки точек для Яндекс Карт;
- адаптивная версия для телефона;
- прозрачные источники и дата редакционной проверки.

Продуктовая защита, критерии приёмки и тест-кейсы описаны в [`docs/PRODUCT.md`](docs/PRODUCT.md).

## Локальный запуск

Требуется Node.js 22.13 или новее.

```bash
npm install
npm run dev
```

## Проверка

```bash
npm run lint
npm test
```

`npm test` сначала делает production build, затем проверяет серверный HTML и ключевые продуктовые требования.

## Публикация

Проект не использует backend, авторизацию, токены или базу данных. Его можно публиковать как обычный статический/edge-сайт на Cloudflare, Vercel или другом хостинге с поддержкой Node-сборки.

Для Tilda практичнее один из двух вариантов:

1. опубликовать этот проект отдельно и дать на него ссылку с Tilda;
2. встроить опубликованную страницу в блок T123 через `iframe`.

Копировать собранный HTML внутрь Tilda целиком не рекомендуется: интерактивные фильтры и стили проще поддерживать в одном исходном проекте.
